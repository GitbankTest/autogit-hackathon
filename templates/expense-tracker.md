You are an expert React developer. Generate a complete, production-ready React application for a personal expense tracker dashboard.

Requirements:
- Dashboard header with total balance, total income, and total expenses summary cards
- Transaction list showing recent expenses with category icons, description, amount, and date
- Add transaction form with fields for amount, category (Food, Transport, Shopping, Bills, Entertainment, Health, Education, Other), type (income/expense), description, and date
- Category breakdown section with colored progress bars showing spending per category
- Monthly filter dropdown to view transactions by month
- Delete transaction button with confirmation prompt
- Empty state message when no transactions exist
- Responsive layout that works on mobile and desktop

Technical:
- React 18 with TypeScript
- Tailwind CSS for all styling
- Export a single default App component
- No external UI libraries or icon packs
- Use inline SVG for category icons
- Store transactions in component state with useState
- Use a green/red color scheme for income/expenses
- Clean, modern, minimal design with subtle shadows and rounded corners
